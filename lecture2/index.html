§ 01

Strategic Form Game의 정의

Strategic-form game (= normal-form game)은 동시에 행동을 선택하는 플레이어들 간의 전략적 상황을 수학적으로 기술한다.

Definition — n-player Strategic Form Game

n-player strategic form game G는 n-tuple {(S₁,u₁),...,(Sₙ,uₙ)}로 정의되며, 세 가지 요소로 구성된다.

① Players

N = {1, 2, ..., n} — 플레이어들의 집합 (비어있지 않음)

② Strategy Space

Sᵢ — 각 플레이어 i ∈ N의 전략 집합 (비어있지 않음)

③ Payoff Function

uᵢ : ∏ₖ∈N Sₖ → ℝ — 전략 프로파일 → 플레이어 i의 보수(실수)

핵심 표기법 (Notation)

표기의미
S := ∏ Sₖ전략 프로파일 전체 집합 (all strategy profiles)
s = (s₁,...,sₙ) ∈ S전략 프로파일 (strategy profile) — 각 플레이어의 전략 조합
S₋ᵢ := ∏ₖ≠ᵢ Sₖ플레이어 i를 제외한 나머지 전략 집합
(sᵢ', s₋ᵢ)플레이어 i만 sᵢ'로 바꾼 프로파일
u : S → ℝⁿ게임 = vector-valued function
💡 핵심 가정 (Hidden Assumptions)
  • 플레이어들은 동시에 행동을 선택한다 (simultaneous play)
  • 모든 보수 구조는 공통지식이다 (all payoffs are common knowledge)
  • 나중에 이 두 가정은 완화된다 (순서·정보 구조 추가)
§ 02

고전 예시 게임들

Prisoners' Dilemma (죄수의 딜레마)

P2: Don't P2: Confess
P1: Don't 5, 5 −1, 6
P1: Confess 6, −1 0, 0
🔑 핵심 인사이트

두 플레이어 모두 Confess가 strictly dominant strategy이다. 따라서 이성적 플레이어들은 항상 (Confess, Confess) = (0,0)을 선택한다. 하지만 이는 Pareto-inefficient — 사회 전체적으로 (5,5)가 더 좋다. 개인 합리성 ≠ 사회 최적.

Battle of the Sexes (남녀 전쟁)

Anne ↓ / Bob → BTS UCL
BTS 4, 1 0, 0
UCL 0, 0 1, 4

함께 있는 것이 혼자인 것보다 좋다. Anne은 BTS를, Bob은 UCL을 선호. 복수 NE 존재 — 조정 문제(coordination problem)의 전형.

Stag Hunt (사슴 사냥 — Rousseau)

H1 ↓ / H2 → Stag Hare
Stag 4, 4 1, 3
Hare 3, 1 2, 2

협력하면 큰 보상(Stag, Stag=4,4), 혼자 사냥하면 토끼만 잡음. 사회적 딜레마 — 신뢰와 협력의 중요성.

Matching Pennies (동전 맞추기)

Anne ↓ / Bob → H T
H +1, −1 −1, +1
T −1, +1 +1, −1

순수전략 NE 없음! Zero-sum 게임. 혼합전략 NE만 존재: σ* = (½H, ½T).

Second-Price Sealed-Bid Auction (2차 가격 경매)

Example: Vickrey Auction
  • n명의 입찰자, 각각 valuation v₁ ≤ ... ≤ vₙ (공통지식)
  • Strategy: Sᵢ ∈ [0, ∞) (입찰 금액)
  • 가장 높은 입찰자가 낙찰, 그러나 2등 입찰가를 지불
uᵢ(sᵢ, s₋ᵢ) = vᵢ − max_{j≠i} sⱼ if sᵢ > max_{j≠i} sⱼ (단독 최고) (1/|W(s)|)(vᵢ − sᵢ) if sᵢ = max_{j≠i} sⱼ (동점) 0 if sᵢ < max_{j≠i} sⱼ (패배)
💡 핵심 결과

2차 가격 경매에서 진가(vᵢ)로 입찰하는 것이 weakly dominant strategy이다. 즉, sᵢ = vᵢ가 지배전략.

§ 03

Mixed Strategies (혼합전략)

Definition — Mixed Strategy

Sᵢ의 원소에 대한 확률 분포(probability distribution)를 혼합전략이라 하며, σᵢ ∈ Δ(Sᵢ)로 표기한다.

Sᵢ가 유한할 때: σᵢ : Sᵢ → [0,1] s.t. ∑_{sᵢ∈Sᵢ} σᵢ(sᵢ) = 1

기대보수 (Expected Payoff)의 확장

uᵢ(σ₁,...,σₙ) := ∑_{s₁∈S₁} ··· ∑_{sₙ∈Sₙ} uᵢ(s₁,...,sₙ) · σ₁(s₁)···σₙ(sₙ) uᵢ(sᵢ, σ₋ᵢ) := ∑_{s₋ᵢ∈S₋ᵢ} uᵢ(sᵢ,s₋ᵢ) ∏_{j≠i} σⱼ(sⱼ) [sᵢ 고정] uᵢ(σᵢ, σ₋ᵢ) := ∑_{sᵢ∈Sᵢ} uᵢ(sᵢ, σ₋ᵢ) · σᵢ(sᵢ) [완전 혼합]

혼합전략 예시: Matching Pennies

Calculation Example

σᵢ(H) = σᵢ(T) = ½ for i = Anne, Bob 일 때

u_Anne(σ_A, σ_B) = (+1)(½)(½) + (−1)(½)(½) + (−1)(½)(½) + (+1)(½)(½) = 0
⚠️ 중요 구별: 상관관계 없음!

혼합전략은 플레이어들 간의 독립적인 확률화를 의미한다.

∏ⁿⱼ₌₁ Δ(Sⱼ) ≠ Δ(∏ⁿⱼ₌₁ Sⱼ) = Δ(S)

좌변(혼합전략 공간)은 우변(전체 결합 분포 공간)의 부분집합이다. 혼합전략은 플레이어 간 상관(correlation)을 허용하지 않는다.

§ 04

Rationality & Common Knowledge

합리성 (Rationality)

각 플레이어는 기대효용을 극대화하기 위해 행동을 선택한다고 가정한다. 하지만 합리성만으로는 게임을 풀기 충분하지 않다.

Definition — Common Knowledge

어떤 사실이 플레이어 그룹 내에서 공통지식(common knowledge)이려면:

  • 모두가 그것을 안다
  • 모두가 모두가 안다는 것을 안다
  • 모두가 모두가 모두가 안다는 것을 안다...
  • 무한히 반복되는 추론 체인
💡 공통지식 vs 상호지식

Mutual knowledge: 모두가 안다 ← 이것만으로는 부족!

Common knowledge: 무한 차수의 추론이 모두 성립 ← 균형·조정·예측에 필수적

파란 눈 섬 주민 예시 (Blue-Eyed Islanders)

Example: Common Knowledge Puzzle

상황: 섬에 파란 눈과 갈색 눈 주민이 있다. 거울이 없어 자신의 눈 색깔을 모른다. 파란 눈임을 알게 되면 자정에 떠나야 한다.

방문객 선언: "이 섬에 파란 눈을 가진 사람이 최소 한 명 있다."

결과: 파란 눈 주민이 정확히 k명이면, k번째 밤에 모두 떠난다.

k명의 파란 눈 주민이 있을 때: - 각자는 (k-1)명을 본다 - "내가 파란 눈이 아니라면 나머지 (k-1)명이 (k-1)번째 밤에 떠날 것" - (k-1)번째 밤에 아무도 안 떠나면 → "나도 파란 눈!" → k번째 밤에 모두 출발
💡 Key Takeaway

방문객의 선언이 새로운 사실을 알려준 게 아니다. 모두가 이미 파란 눈 주민을 봤다. 핵심은 공통지식으로 전환시킨 것이다. 공통지식이 없으면 아무도 떠나지 않는다.

§ 05

Dominance (지배전략)

Strict Dominance (강한 지배)

Definition — Strictly Dominates

전략 sᵢ'이 sᵢ''를 strictly dominate한다 ⟺

∀s₋ᵢ ∈ S₋ᵢ: uᵢ(sᵢ', s₋ᵢ) > uᵢ(sᵢ'', s₋ᵢ)

즉, 상대방의 어떤 전략에 대해서도 sᵢ'이 sᵢ''보다 항상 더 좋다.

Definition — Strictly Dominant Strategy

sᵢ ∈ Sᵢ가 strictly dominant strategy이다 ⟺ sᵢ가 다른 모든 전략 sᵢ'' ∈ Sᵢ \ {sᵢ}를 strictly dominate.

Lemma

플레이어 i가 strictly dominant strategy를 가지면, arg max_{sᵢ} uᵢ(sᵢ, s₋ᵢ)는 s₋ᵢ에 독립적이며 유일하다.

혼합전략에 의한 지배

Example: Pure Strategy로 지배 안 되는 경우
Anne ↓ / Bob →LR
U3, 00, 0
M0, 03, 0
D1, 01, 0

D는 U나 M 어느 하나로도 strictly dominated되지 않는다. 그러나 σ = (½U, ½M)은 D를 strictly dominate!

σ = (½, ½, 0): u(σ, L) = ½(3) + ½(0) = 1.5 > 1 = u(D, L) ✓ u(σ, R) = ½(0) + ½(3) = 1.5 > 1 = u(D, R) ✓
Definition — Dominated by Mixed Strategy

순수전략 sᵢ'가 혼합전략 σᵢ에 의해 strictly dominated된다 ⟺

uᵢ(sᵢ', s₋ᵢ) < uᵢ(σᵢ, s₋ᵢ) ∀s₋ᵢ ∈ S₋ᵢ
§ 06

IDSDS (반복적 지배전략 삭제)

Definition — Iterated Deletion of Strictly Dominated Strategies

다음 절차를 반복한다:

  1. 현재 게임에서 strictly dominated된 모든 전략을 삭제한다.
  2. 남은 게임에서 strictly dominated 전략이 있는가?
    → Yes: 1로 돌아간다.   → No: 종료.
💡 Dominance Solvable

이 절차가 유일한 결과로 수렴하는 게임을 dominance solvable (지배 가능 게임)이라 한다.

IDSDS 예시

IDSDS Step-by-Step
Anne ↓ / Bob →XYZ
A3, 30, 50, 4
B0, 03, 11, 2
  1. Bob의 관점: X vs Y vs Z 비교.
    u_Bob(Y, A)=5 > u_Bob(X, A)=3, u_Bob(Y, B)=1 > u_Bob(X, B)=0
    → X is strictly dominated by Y → X 삭제
  2. Y vs Z (X 삭제 후):
    u_Bob(Y, A)=5 > u_Bob(Z, A)=4, u_Bob(Y, B)=1 < u_Bob(Z, B)=2
    → Y가 Z를 strictly dominate하지 않음. Z가 Y를 strictly dominate하지 않음.
  3. Anne의 관점 (X 삭제 후):
    A vs B: u(A, Y)=0 < u(B, Y)=3, u(A, Z)=0 < u(B, Z)=1
    → A is strictly dominated by B → A 삭제
  4. 남은 게임 {B} × {Y, Z}:
    u_Bob(Y, B)=1 < u_Bob(Z, B)=2
    → Y is strictly dominated by Z → Y 삭제
  5. 결과: (B, Z) 유일! — Dominance Solvable
🔑 IDSDS의 순서 불변성

IDSDS에서는 어떤 순서로 strictly dominated 전략을 제거해도 동일한 결과가 나온다. (Order doesn't matter for strict dominance)

§ 07

Weak Dominance (약한 지배)

Definition — Weakly Dominated

전략 sᵢ'가 σᵢ에 의해 weakly dominated된다 ⟺

∀s₋ᵢ ∈ S₋ᵢ: uᵢ(sᵢ', s₋ᵢ) ≤ uᵢ(σᵢ, s₋ᵢ) ∃s'₋ᵢ ∈ S₋ᵢ: uᵢ(sᵢ', s'₋ᵢ) < uᵢ(σᵢ, s'₋ᵢ) (적어도 하나는 엄격)

즉, 모든 경우에 최소한 같거나 나쁘고, 적어도 하나의 경우에는 엄격히 나쁘다.

IDWDS (Iterated Deletion of Weakly Dominated Strategies) — 왜 문제인가?

⚠️ 순서 의존성 (Order Dependence)

IDWDS에서는 삭제 순서에 따라 결과가 달라진다!

Example: Order Matters
Anne ↓ / Bob →LR
U1, 10, 0
M1, 12, 1
D0, 02, 1
경로 1

U 삭제 (weakly dominated by M) → L 삭제 → (M,R) 또는 (D,R)

경로 2

D 삭제 (weakly dominated by M) → R 삭제 → (U,L) 또는 (M,L)

결론: IDWDS는 순서에 따라 전혀 다른 결과를 낳으므로, 분석 도구로서 신뢰성이 낮다.

§ 08

Nash Equilibrium (순수전략 NE)

많은 게임에서 IDSDS로도 유일한 균형을 찾지 못한다. 더 강력한 균형 개념이 필요하다.

Definition — Nash Equilibrium (Pure Strategies)

전략 프로파일 s* ∈ S가 Nash equilibrium in pure strategies이다 ⟺

∀i ∈ N, ∀sᵢ' ∈ Sᵢ: uᵢ(sᵢ*, s*₋ᵢ) ≥ uᵢ(sᵢ', s*₋ᵢ)

즉, 어떤 플레이어도 상대방의 전략이 고정되어 있을 때 단방향으로 이탈할 인센티브가 없다.

Best Response를 통한 정의

Definition — Best Response

주어진 σ₋ᵢ에 대해, sᵢ가 best response이다 ⟺

∀sᵢ' ∈ Sᵢ: uᵢ(sᵢ, σ₋ᵢ) ≥ uᵢ(sᵢ', σ₋ᵢ)

Best response 집합: BRᵢ(σ₋ᵢ)

Definition — NE via Best Response

s*가 NE이다 ⟺ ∀i ∈ N: sᵢ* ∈ BRᵢ(s*₋ᵢ)

즉, 모든 플레이어가 동시에 서로의 best response인 전략 프로파일.

NE 판별: 화살표(best response) 방법

예시 게임에서 NE 찾기 (4×4 Matrix)
Anne ↓ / Bob →WXYZ
A5, 23, 60, 22, 2
B7, 67, 73, 22, 0
C3, 22, 33, 12, 0
D2, 19, 22, 02, 1

답: (D, X) — 각 플레이어의 best response 화살표가 (D,X)에서 만남.

Anne given Bob plays X: u(A,X)=3, u(B,X)=7, u(C,X)=2, u(D,X)=9 → BR = D ✓ Bob given Anne plays D: u(D,W)=1, u(D,X)=2, u(D,Y)=0, u(D,Z)=1 → BR = X ✓
💡 시험 필수 체크!

NE를 찾을 때 각 플레이어의 전략을 반드시 모두 명시해야 한다! "NE는 X입니다"처럼 플레이어 중 하나의 전략만 답하면 오답.

NE의 해석 4가지

#해석
1게임이 어떻게 플레이될지에 대한 예측
2실제 플레이와 비교하기 위한 벤치마크
3사전 소통, 사회 규범, 반복 등에 의해 안정화된 전략 프로파일
4"Rational expectations" 하에서 자기실현적 믿음
§ 09

Nash Equilibrium (혼합전략 MSNE)

Definition — MSNE (Mixed Strategy Nash Equilibrium)

(σ₁*, ..., σₙ*)가 MSNE ⟺

∀i ∈ N, ∀σᵢ' ∈ Δ(Sᵢ): uᵢ(σᵢ*, σ*₋ᵢ) ≥ uᵢ(σᵢ', σ*₋ᵢ)

등가 조건: σᵢ* ∈ BRᵢ(σ*₋ᵢ) for all i

💡 MSNE의 핵심 성질
  • 순수전략 NE는 MSNE의 특수 케이스이다 (Pure NE ⊆ MSNE)
  • 플레이어가 양의 확률로 사용하는 모든 순수전략은 같은 기대보수를 가져야 한다 (indifference condition)
  • 사용하지 않는 순수전략의 기대보수는 사용하는 것보다 작거나 같아야 한다

MSNE 계산: Matching Pennies

MSNE 계산 — Indifference Condition 활용

pA = Pr(H_Anne), pB = Pr(H_Bob)

Anne's Best Response:

pB > ½ :BRA = {H} (pA = 1)
pB = ½ :BRA = [0,1] (indifferent)
pB < ½ :BRA = {T} (pA = 0)

MSNE: ((pA, 1−pA), (pB, 1−pB)) = ((½, ½), (½, ½))

두 BR 함수의 교점이 MSNE가 된다.

MSNE 계산: Battle of Partners 예시

MSNE Full Calculation
1 ↓ / 2 →LR
T2, 10, 0
B0, 01, 1

PSNE: (T, L) (B, R)

MSNE 계산 (pT = Pr(T), pL = Pr(L)):

P1의 무차별 조건: u₁(T, pL) = u₁(B, pL) 2pL = 1(1 − pL) → 3pL = 1 → pL = 1/3 P2의 무차별 조건: u₂(L, pT) = u₂(R, pT) 1·pT = 1·(1 − pT) → pT = 1/2 MSNE: ((½, ½), (⅓, ⅔))

전체 NE 목록: (T,L), (B,R), ((½, ½), (⅓, ⅔)) — 총 3개 (홀수!)

§ 10

NE의 개수 — Wilson-Harsanyi 정리

Theorem (Wilson, Harsanyi)

"거의 모든" 유한 게임에서 NE의 개수는 홀수이다.

즉, 짝수 개의 NE를 가지는 유한 게임의 집합은 measure zero이다.

💡 시험 필수 확인 사항

문제에서 NE를 모두 찾았다면, 그 개수가 홀수인지 확인하는 것이 좋다! 짝수가 나왔다면 빠뜨린 NE가 있을 가능성이 높다.

예시 확인
  • Prisoner's Dilemma: NE = {(C,C)} → 1개 (홀수) ✓
  • Matching Pennies: NE = {((½,½),(½,½))} → 1개 (홀수) ✓
  • Battle of Partners (2×2): NE = {(T,L), (B,R), MSNE} → 3개 (홀수) ✓
§ 11

Refinements — Trembling Hand Perfection

많은 게임에서 NE가 복수로 존재한다. 정련(refinement)을 통해 불합리한 균형을 제거한다.

Definition — Trembling Hand Perfect Equilibrium (THPE)

전략 σᵢ가 completely mixed이다 ⟺ σᵢ(sᵢ) > 0 for all sᵢ ∈ Sᵢ

전략 프로파일 σ가 THPE이다 ⟺ 완전 혼합 전략의 수열 σᵏ가 존재하여:

(i) lim_{k→∞} σᵏ = σ (수렴) (ii) σᵢ ∈ BRᵢ(σᵏ₋ᵢ) for all k (각 k에서 best response)
Theorem (Selten)

모든 유한 게임에서 Trembling Hand Perfect Equilibrium이 존재한다.

THPE 예시: (R,R) 제거

Example: Ruling out (R,R)
Anne ↓ / Bob →LR
L1, 10, −3
R−3, 00, 0

두 개의 순수전략 NE: (L,L)과 (R,R)

(L,L) — THPE ✓

Anne이 실수로 R을 ε 확률로 선택할 때:
Bob의 기대보수: L → (1−ε), R → −3(1−ε)
(1−ε) > −3(1−ε) for all ε>0 → Bob's BR = L
대칭적으로 Anne's BR = L
→ (L,L)은 THPE

(R,R) — Not THPE ✗

Anne이 실수로 L을 ε 확률로 선택할 때:
Bob의 기대보수: L → ε, R → −3ε
ε > −3ε → Bob's BR = L (not R)
→ (R,R)은 THPE가 아님

🔑 THPE의 직관

각 플레이어가 아주 작은 확률로 "손 떨림(trembling)"에 의해 실수를 한다고 가정한다. 이 미세한 섭동(perturbation) 하에서도 여전히 best response인 균형만이 THPE로 살아남는다. Weakly dominated 전략을 포함한 균형은 제거된다.

§ 12

Cournot's Model of Quantity Competition

Setup: Cournot Duopoly
  • 두 기업 1, 2가 동질 재화(homogeneous good)를 생산
  • 역수요 함수: P(Q) = max{a − Q, 0}, where Q = q₁ + q₂
  • 비용 함수: C(qᵢ) = c·qᵢ, where 0 < c < a
  • 두 기업이 동시에 생산량을 결정

게임 구조

Players: {1, 2} Strategies: Sᵢ = {qᵢ : qᵢ ≥ 0} Payoff: uᵢ(q₁, q₂) = (P(q₁+q₂) − c)·qᵢ = [(a − c) − (qᵢ + q₋ᵢ)]·qᵢ (a−Q ≥ 0인 경우)

Best Response 도출

u₁(q₁, q₂) = [(a−c) − (q₁ + q₂)]·q₁ 을 q₁에 대해 최대화 → strictly concave in q₁ → F.O.C.로 유일한 최대값 ∂u₁/∂q₁ = (a−c) − 2q₁ − q₂ = 0 BR₁(q₂) = (a−c−q₂)/2 대칭적으로: BR₂(q₁) = (a−c−q₁)/2

NE 계산

NE 조건: q₁ = BR₁(q₂) and q₂ = BR₂(q₁) 연립방정식: q₁ = (a−c−q₂)/2 q₂ = (a−c−q₁)/2 → 2q₁ + q₂ = a−c = q₁ + 2q₂ → q₁ = q₂ (대칭성) → q* = (a−c)/3 유일한 NE: q₁* = q₂* = (a−c)/3
💡 비교: Monopoly vs Cournot vs Competitive
시장구조총생산량 Q가격 P특징
독점(a−c)/2(a+c)/2최소 생산
Cournot 복점2(a−c)/3(a+2c)/3중간
완전경쟁a−cc최대 생산

Q. c₁ ≠ c₂인 경우

BR₁(q₂) = (a − c₁ − q₂) / 2 BR₂(q₁) = (a − c₂ − q₁) / 2 NE: q₁* = (a − 2c₁ + c₂) / 3 q₂* = (a − 2c₂ + c₁) / 3
§ 13

Bertrand's Model of Price Competition

Setup: Bertrand Competition

Cournot와 동일한 수요·비용 구조이나, 이번에는 가격을 동시에 설정한다. 소비자는 가장 낮은 가격의 기업에서 구매.

u₁(p₁, p₂) = (p₁ − c)(a − p₁) if p₁ < p₂ (독점 판매) (p₁ − c)(a − p₁)/2 if p₁ = p₂ (동점 → 반반) 0 if p₁ > p₂ (판매 없음)

NE 도출 — 유일한 균형: p₁* = p₂* = c

증명 (귀류법): Case 1: p₁ > p₂ > c → Firm 1의 payoff = 0 (아무것도 못 팜) → p₁' = p₂ − ε로 낮추면 양의 payoff 획득 → 이탈 인센티브 존재 Case 2: p₁ = p₂ > c → Firm 1의 payoff = (p₁−c)(a−p₁)/2 → p₁' = p₁ − ε로 낮추면 payoff ≈ (p₁−ε−c)(a−p₁+ε) > 반분한 것 → 이탈 인센티브 존재 Case 3: p* = c (제안) → Firm 1의 payoff = 0 (c에서는 이윤 없음) → 올리면 0, 내리면 음수 → 이탈 인센티브 없음 ✓ ∴ 유일한 NE: p₁* = p₂* = c
🔑 Bertrand Paradox

단 두 개의 기업만 있어도 가격이 한계비용(c)까지 하락한다! 이는 완전경쟁과 동일한 결과다. Cournot와의 대조: 수량 경쟁 → 독점 수준 이상의 이윤, 가격 경쟁 → 이윤 = 0.

§ 14

Hotelling's Spatial Competition (1929)

Setup: Spatial Competition
  • 소비자 x ∈ [0,1]이 균등분포 (uniform distribution)
  • 판매자 A는 위치 0, 판매자 B는 위치 1에 고정
  • 소비자 x의 이동 비용: A까지 x, B까지 (1−x)
  • 재화의 효용: sA = sB = s (동질적)
  • 각 판매자가 가격 pA, pB를 동시에 설정
소비자 x가 A를 선택 ⟺ s − pA − x ≥ s − pB − (1−x) ⟺ pB − pA + ½ ≥ x ⟺ x ≤ (pB − pA + 1)/2 = x̃ (무차별 소비자)

Best Response 도출

πA(pA | pB) = pA × [(pB − pA + 1)/2] (pA의 strictly concave 함수) F.O.C.: ∂πA/∂pA = (pB − 2pA + 1)/2 = 0 BRA(pB) = (pB + 1) / 2 ← 전략적 보완재! (strategic complementarity) 대칭적으로: BRB(pA) = (pA + 1) / 2

Nash Equilibrium

NE 조건: pA = BRA(pB), pB = BRB(pA) 대칭 NE: pA = pB = p* p* = (p* + 1)/2 → 2p* = p* + 1 → p* = 1 균형 소비자 경계: x* = (p* − p* + 1)/2 = 1/2 (시장 이등분) 균형 이윤: πA = πB = p* × (1/2) = 1/2 소비 조건: x*의 소비자가 비음의 잉여를 가져야 함 s − p* − x* ≥ 0 → s − 1 − 1/2 ≥ 0 → s ≥ 3/2
🔑 결과 해석

위치가 고정된 경우, 두 판매자는 모두 가격 p* = 1을 책정하고 시장을 정확히 절반씩 나눈다. 가격 경쟁에서도 수송비(transport cost)가 독점력의 원천이 된다.

Q1. s < 3/2인 경우: 일부 소비자가 시장에서 이탈 → 분석 복잡해짐

Q2. sA ≠ sB인 경우: 가격이 비대칭적으로 결정됨

§ 15

Platform Competition — Median Voter Theorem

Setup: Median Voter
  • 정책 공간: [0,1] (진보 → 보수)
  • 유권자의 이상적 위치: 균등분포 over [0,1]
  • 유권자는 자신의 이상점에 가장 가까운 후보에게 투표 (무차별 → 각각 ½ 확률)
  • 두 후보 A, B가 동시에 정책 위치 sᵢ ∈ [0,1]를 선택
  • 보수 = 얻은 득표율 (정책 선호 없음, 오직 득표 극대화)

Best Response 도출

s₁ < s₂ 인 경우: Candidate 1의 득표율: s₁ + (s₂ − s₁)/2 = (s₁ + s₂)/2 u₁(s₁, s₂)는 s₁ < s₂이면 s₁ 증가에 따라 증가 s₁ > s₂이면 s₁ 증가에 따라 감소 → s₁ = s₂에서 최적 (상대방을 모방하는 것이 최선) 따라서, s₁ = s₂ in any NE

Nash Equilibrium: s* = ½

증명: s* > ½ 이면: Candidate 2는 s₂' = s* − ε 선택 시 더 많은 득표 (중간값 위쪽에서 오른쪽을 독점) s* < ½ 이면: 대칭적 논리로 NE 불가 s* = ½ 이면: u₁(s₁, ½)는 s₁ = ½에서 유일한 최대값 ✓ u₂(½, s₂)도 s₂ = ½에서 유일한 최대값 ✓ 유일한 NE: s₁* = s₂* = ½
Theorem — Median Voter Theorem

양당 경쟁에서 두 후보가 득표율 극대화를 추구할 때, 균형은 중위 투표자(median voter)의 선호위치(½)에서 동일하게 정책을 채택하는 것이다.

🔑 정치·경제적 함의

이 결과는 실제 선거에서 두 주요 정당이 종종 중도 수렴(centrist convergence) 하는 경향을 설명한다. 그러나 현실에서는 정책 선호(policy preferences), 당원 기반(base voters), 이념(ideology) 등으로 인해 완전한 수렴이 일어나지 않는다.
